### Want to use this as a template?

1. Fork this repo
2. Clone it: `git clone https://github.com/YOUR_USERNAME/yaninatrekhleb.github.io.git`
3. Replace the content in `index.html` with your own
4. Add your photo and update the `CNAME` file with your domain
5. Push your changes:
   ```
   git add .
   git commit -m "My portfolio"
   git push origin main
   ```
6. Your site will deploy automatically via GitHub Pages
